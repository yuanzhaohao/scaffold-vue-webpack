## A scaffold built by vue and webpack

一个简单的vue工程脚手架

同时还使用了手淘的Flexible跨终端适配高清化布局方案，这是一套手机淘宝h5页面使用的布局方案，详情可看[使用Flexible实现手淘H5页面的终端适配](https://github.com/amfe/article/issues/17)

### how to start

To build the examples locally, run:

```
npm install
npm start
```
### how to build
```
npm run build
```
